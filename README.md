# 🎮 Number Guessing Game

## 📌 Description
This is a simple C program where the computer generates a random number and the user tries to guess it.

---

## ⚙️ Features
- Generates random number using `rand()`
- Uses `srand()` for different output each time
- User guesses the number
- Displays correct or wrong result

---

## ▶️ How It Works
1. Computer generates a number between 1 and 10  
2. User enters a guess  
3. Program checks the guess  
4. Displays result
